Controls Documentation

Jump -- 'W' or 'Space'

Sneak/Crouch -- 'S' or 'Ctrl'

Left -- 'A'

Right -- 'D'

Hotswitch Toggle -- 'Q'

Hotswitch Key -- 'F'

Attack -- 'LMB'

Interact -- 'E'
